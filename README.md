Songlist
========

## Desarrollo

Dependencias: ruby/bundler/rake, npm

```bash
npm install
bower install
bundle install
```

Correr servidor:
`rails server`

Correr tests:
`RAILS_ENV=test bundle exec rake spec:javascript`
